第一章：认识我们的隐私保护利器

1.1 豆本豆奶——隐私保护的"智能管家"

豆本豆奶绝非普通的网络代理工具，它是基于V2Ray核心开发的一款"隐私增强型"客户端。如果说V2Ray是功能强大的发动机，那么豆本豆奶就是精心设计的汽车外壳——它将复杂的配置过程简化，让普通用户也能轻松驾驭这项尖端技术。

1.2 V2Ray——网络自由的"瑞士军刀"

V2Ray堪称代理技术领域的"变形金刚"。它不像传统VPN那样只能提供单一的隧道加密，而是支持多种协议和传输方式的动态组合。这种设计理念使其具备了惊人的适应性：

  • 协议动物园：支持VMess、Shadowsocks、Socks等主流协议

  • 路由魔术师：基于域名的精细路由控制，实现国内外流量智能分流

特别值得一提的是其"多入口多出口"架构，用户可以在不同场景下使用不同的协议组合，这种设计极大提高了被封锁的难度。

2.1 跨平台部署指南

FAQ

V2Ray 的 ws+tls 传输模式有什么优点？
ws+tls 模式利用 WebSocket 伪装成普通 HTTPS 流量，使得防火墙难以区分代理与正常访问。该模式兼容性强，适合部署在支持 Nginx 或 Caddy 的 Web 服务器上。
V2Ray 如何实现分流规则？
V2Ray 使用 routing 模块实现分流，可根据目标域名、IP、端口或协议类型决定流量走向。用户可自定义多个 outbound，并通过规则实现国内直连、国外代理等智能路由功能。
Potatso Lite 可以管理多少个节点？
Potatso Lite 可以管理数十个节点，适合轻量用户使用。虽然不支持批量自动更新，但可以手动添加和删除节点，结合策略组实现基本分流和管理功能。
Hysteria2 客户端延迟测试值不稳定怎么办？
可通过调整 Hysteria2 的 `recv_window` 与 `send_window` 参数改善传输性能，同时确保服务器带宽充足。若使用移动网络，建议启用 BBR 拥塞控制并降低测速频率以减少波动。
Nekobox 是否支持多客户端节点管理？
支持。Nekobox 可以独立管理 Shadowsocks、Clash 等客户端节点，互不干扰。结合策略组，可实现多客户端流量分流和网络优化，提高节点管理效率和使用便利性。
Matsuri 的延迟测试准确吗？
Matsuri 的延迟测试使用 ICMP 与 TCP 双重检测方式，可较准确反映节点实际响应时间。不过受网络波动影响，建议多次测试并结合速度测试结果综合判断。
如何在 Sing-box 中设置多个 outbound 实现策略路由？
在 sing-box 的配置中定义多个 outbound 对象并为 routing 模块配置 rules，将不同目的地或应用流量指向对应 outbound，从而实现分流、负载均衡或故障转移等复杂路由策略。
Nekobox 是否支持跨客户端节点管理？
支持。Nekobox 可以独立管理 Shadowsocks、Clash 等客户端节点，互不干扰。结合策略组，可实现多客户端流量分流和网络优化，提高节点管理效率和使用便利性。
Clash 如何通过策略组实现多节点负载均衡？
Clash 支持策略组功能，可以根据节点延迟、地区或用途自动选择最佳节点处理流量，实现多节点负载均衡。有效提高访问速度和稳定性，保证网页浏览、视频播放和游戏流畅。
Netch 支持游戏加速吗？
Netch 专为游戏加速设计，支持 UDP 转发和 TAP 驱动，可有效降低延迟。用户可以为特定游戏添加路由规则，实现分流加速，从而提升国际游戏体验。
